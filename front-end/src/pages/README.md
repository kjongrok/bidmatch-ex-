# Pages

라우팅 단위 화면을 보관합니다.

## 예정 화면

- `DashboardPage`: 사용자 대시보드
- `NoticeSearchPage`: 공고 검색/목록
- `NoticeDetailPage`: 공고 상세
- `MatchRulesPage`: 관심 조건 관리
- `DeadlineCalendarPage`: 마감일 달력
- `NotificationHistoryPage`: 알림 이력
- `ProfileSettingsPage`: 프로필/기업 정보 설정
- `AdminDashboardPage`: 관리자 대시보드

## 개발 방향

초기 목업은 `src/App.jsx`에 있지만, 실제 개발이 시작되면 화면별 파일로 분리합니다.
